Running JACK on Linux

UTC 2025-07-15th, 2025

So, about a month ago I spent much longer than I'd ever like to admit trying to get the Steam version of JACK to run on my laptop running Fedora. If you don't know, JACK is an alternative to Valve's original Hammer map editor for games running on the Goldsrc engine. I installed maybe a million old packages, scouring the web for some semblance of help, and I got it working, but long story short, it stopped working again.

Fast forward to now, I decided to screw with it some more, and figured out the world's easiest solution, as detailed below:

  1. Boot up Steam
  2. Right-click on JACK and select Preferences
  3. Go to Compatability and click the checkbox that says Force the use of a specific Steam Play compatability tool
  4. Lastly, select Proton Hotfix in the dropdown

And JACK should boot up like nothing. I'm assuming the Linux version just isn't very well maintained, so using Proton is the way to go. If you're having issues with the free version, I'd honestly just recommend waiting for a Steam sale and picking it up for a decent price then. Happy piping!
